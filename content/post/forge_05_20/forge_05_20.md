### 文献综述
近年来，因果学习因其能够揭示现象背后的本质关系与机制，已成为人工智能领域的研究热点。传统的因果学习方法在处理高维、非结构化数据以及未观测混杂因素时面临挑战，而深度学习为其提供了强大的表征与建模能力，由此催生了“深度因果学习”这一前沿交叉领域。本综述将围绕因果表征学习、因果发现、因果推断及其在下游任务中的应用，对相关文献进行总结。
Deng等人(2025)[1]对深度因果学习进行了一个全面的综述，系统性地回顾了深度学习如何从**表征、发现和推断**三个维度赋能因果学习，以解决传统方法的局限性。Schölkopf等人(2021)[2]则从相反的视角，探讨了因果关系如何促进机器学习在迁移和泛化上的能力，并提出了因果表征学习作为连接两个领域的核心问题。
在**因果表征学习**方面，核心目标是学习能够反映数据背后因果机制的不变表征。Arjovsky等人(2020)[40]提出了不变风险最小化（IRM），通过学习在不同训练分布中不变的表示来实现分布外泛化，建立了不变性与因果结构的理论联系。Ahuja等人(2022)[41]则探索了干预数据在因果表征学习中的强大作用，证明了在完美干预下可以识别出潜在的因果因素。You等人(2026)[22]提出的DRLCET模型，将变分自编码器（VAE）与图注意力网络（GAT）结合，通过非线性结构因果模型（SCM）更新因果结构并传递因果效应，以学习解耦表征。You等人(2026)[21]还提出了CRLTGA模型，利用三元组网络和图注意力机制聚合因果信息，学习更具因果效应的表征。Yang等人(2021)[42]开发的CausalVAE，通过一个因果层将独立的外生因子转化为符合因果结构的内生因子，实现了语义可解释的因果表征学习和反事实数据生成。Zhou等人(2025)[5]则提出了GeAE，一个基于图自编码器的自编码器框架，用于学习跨域不变的因果表征以服务于迁移学习。
在具体的**下游任务应用**中，因果学习被广泛用于提升模型的鲁棒性、可解释性和泛化能力。在**推荐系统**领域，Ding等人(2024)[11][12]提出了因果增量图卷积（IGC）方法，解决了图神经网络（GNN）模型在增量更新时因丢弃旧图结构而导致的准确率下降问题，通过碰撞效应蒸馏（CED）利用因果推断连接新数据与不活跃节点。在**计算机视觉**领域，因果干预被用于消除伪相关。Liu等人(2023)[4]提出了跨模态因果关系推理框架（CMCIR），利用前门和后门因果干预来解开视觉问答中的跨模态伪相关。Li等人(2023)[18]提出的Causal-ViT和Wang等人(2023)[43]提出的Causal-SETR，都通过构建结构因果模型（SCM）并对全局上下文进行因果干预，消除了ViT模型中的混杂偏差，提升了图像识别和语义分割的鲁棒性。在**遥感图像分析**中，Zhu等人(2024)[13]、Zhao等人(2026)[14]和Shi等人(2024)[15]分别从特征解耦、对象级因果图构建和预训练偏差校正的角度，利用因果干预和反事实推断学习不变特征，有效应对了遥感图像中的域偏移和特征混淆问题。Yu等人(2024)[30]则提出 \mathbf I^3 -CI 学习范式，通过图像内和图像间对比学习来消除偏置。在**视频理解**任务中，Zhang等人(2025)[17]针对长尾分布和伪相关，提出了CICR框架，通过构建SCM识别混淆因子并进行干预和反事实推理，提升了视频动作识别的稳健性。Hu等人(2025)[23]提出的VC-Mamba，通过因果一致性约束来增强视频中隐含信息的理解。
因果学习在**工业与工程**领域也展现出巨大潜力。Zhang等人(2026)[8]提出了因果图时空自编码器（CGSTAE），用于工业过程监控，通过学习变量间的因果图来进行故障检测。Zhou等人(2024)[7]开发了CausalKGPT，一个因果知识增强的大语言模型，通过构建结构因果图来消除伪关联，为航空航天产品制造中的质量问题分析提供可信赖的引导。Zhong等人(2025)[20]则提出了TrafficRiskGPT系统，将因果推断与思维链推理注入大语言模型，用于交通场景风险评估，有效降低了模型幻觉和推理延迟。
此外，一些研究专注于**因果框架下的特定技术挑战**。例如，为了构建稳定且可解释的注意力机制，Hu等人(2025)[6]提出了SEAT和SEAT++，通过对注意力分布施加约束，使其在保持解释性的同时，对随机性和扰动具有鲁棒性。Ma等人(2025)[9]提出了基于互信息的多标签因果特征选择方法（MI-MCF），利用互信息高效地构建马尔可夫毯。Liang等人(2025)[10]开发的DVGNN模型，通过扩散模型推断动态图链接概率并重构因果图，增强了时空预测的可解释性和对不确定性的鲁棒性。
综上所述，当前研究正积极探索将因果推断的理论（如SCM、干预、反事实推理）与深度学习模型（如VAE、GNN、ViT、LLM）深度融合。该领域的核心趋势在于，通过发掘数据背后的因果结构来学习不变表征，从而克服机器学习模型在面对分布外数据、伪相关和缺乏可解释性时的固有问题。未来的研究方向将聚焦于更复杂的因果结构学习、反事实样本的高效生成以及因果理论在更多现实任务中的应用。

| **Common Features** | 绝大多数研究旨在**提升模型的泛化性、鲁棒性或可解释性**。核心方法是将**因果推断的原则（如不变性、干预）与深度学习模型**（如VAE、GNN、Transformer）相结合。研究重心从关联驱动的预测转向**因果驱动的理解与决策**。 | | |
| **Distinctive Features** | 具体任务多样（CV、NLP、推荐、工业）。因果的作用方式不同：有的用于**表征学习**（学习不变因果特征），有的用于**结构发现**（构建因果图），有的用于**去偏/推理**（干预、反事实）。数据假设不同：部分假设仅有观测数据，部分依赖**干预数据**。模型基础结构各异：基于VAE、GNN、Transformer、Mamba等不同架构。 | | |
