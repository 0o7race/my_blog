因果时序模型文献综述

因果时序模型综述

2026-05-20
    • 文献综述
    • 文献矩阵

文献综述

近年来，因果学习因其能够揭示现象背后的本质关系与机制，已成为人工智能领域的研究热点。传统的因果学习方法在处理高维、非结构化数据以及未观测混杂因素时面临挑战，而深度学习为其提供了强大的表征与建模能力，由此催生了“深度因果学习”这一前沿交叉领域。本综述将围绕因果表征学习、因果发现、因果推断及其在下游任务中的应用，对相关文献进行总结。 Deng等人(2025)[1]对深度因果学习进行了一个全面的综述，系统性地回顾了深度学习如何从表征、发现和推断三个维度赋能因果学习，以解决传统方法的局限性。Schölkopf等人(2021)[2]则从相反的视角，探讨了因果关系如何促进机器学习在迁移和泛化上的能力，并提出了因果表征学习作为连接两个领域的核心问题。 在因果表征学习方面，核心目标是学习能够反映数据背后因果机制的不变表征。Arjovsky等人(2020)[40]提出了不变风险最小化（IRM），通过学习在不同训练分布中不变的表示来实现分布外泛化，建立了不变性与因果结构的理论联系。Ahuja等人(2022)[41]则探索了干预数据在因果表征学习中的强大作用，证明了在完美干预下可以识别出潜在的因果因素。You等人(2026)[22]提出的DRLCET模型，将变分自编码器（VAE）与图注意力网络（GAT）结合，通过非线性结构因果模型（SCM）更新因果结构并传递因果效应，以学习解耦表征。You等人(2026)[21]还提出了CRLTGA模型，利用三元组网络和图注意力机制聚合因果信息，学习更具因果效应的表征。Yang等人(2021)[42]开发的CausalVAE，通过一个因果层将独立的外生因子转化为符合因果结构的内生因子，实现了语义可解释的因果表征学习和反事实数据生成。Zhou等人(2025)[5]则提出了GeAE，一个基于图自编码器的自编码器框架，用于学习跨域不变的因果表征以服务于迁移学习。 在具体的下游任务应用中，因果学习被广泛用于提升模型的鲁棒性、可解释性和泛化能力。在推荐系统领域，Ding等人(2024)[11][12]提出了因果增量图卷积（IGC）方法，解决了图神经网络（GNN）模型在增量更新时因丢弃旧图结构而导致的准确率下降问题，通过碰撞效应蒸馏（CED）利用因果推断连接新数据与不活跃节点。在计算机视觉领域，因果干预被用于消除伪相关。Liu等人(2023)[4]提出了跨模态因果关系推理框架（CMCIR），利用前门和后门因果干预来解开视觉问答中的跨模态伪相关。Li等人(2023)[18]提出的Causal-ViT和Wang等人(2023)[43]提出的Causal-SETR，都通过构建结构因果模型（SCM）并对全局上下文进行因果干预，消除了ViT模型中的混杂偏差，提升了图像识别和语义分割的鲁棒性。在遥感图像分析中，Zhu等人(2024)[13]、Zhao等人(2026)[14]和Shi等人(2024)[15]分别从特征解耦、对象级因果图构建和预训练偏差校正的角度，利用因果干预和反事实推断学习不变特征，有效应对了遥感图像中的域偏移和特征混淆问题。Yu等人(2024)[30]则提出 \mathbf I^3 -CI 学习范式，通过图像内和图像间对比学习来消除偏置。在视频理解任务中，Zhang等人(2025)[17]针对长尾分布和伪相关，提出了CICR框架，通过构建SCM识别混淆因子并进行干预和反事实推理，提升了视频动作识别的稳健性。Hu等人(2025)[23]提出的VC-Mamba，通过因果一致性约束来增强视频中隐含信息的理解。 因果学习在工业与工程领域也展现出巨大潜力。Zhang等人(2026)[8]提出了因果图时空自编码器（CGSTAE），用于工业过程监控，通过学习变量间的因果图来进行故障检测。Zhou等人(2024)[7]开发了CausalKGPT，一个因果知识增强的大语言模型，通过构建结构因果图来消除伪关联，为航空航天产品制造中的质量问题分析提供可信赖的引导。Zhong等人(2025)[20]则提出了TrafficRiskGPT系统，将因果推断与思维链推理注入大语言模型，用于交通场景风险评估，有效降低了模型幻觉和推理延迟。 此外，一些研究专注于因果框架下的特定技术挑战。例如，为了构建稳定且可解释的注意力机制，Hu等人(2025)[6]提出了SEAT和SEAT++，通过对注意力分布施加约束，使其在保持解释性的同时，对随机性和扰动具有鲁棒性。Ma等人(2025)[9]提出了基于互信息的多标签因果特征选择方法（MI-MCF），利用互信息高效地构建马尔可夫毯。Liang等人(2025)[10]开发的DVGNN模型，通过扩散模型推断动态图链接概率并重构因果图，增强了时空预测的可解释性和对不确定性的鲁棒性。 综上所述，当前研究正积极探索将因果推断的理论（如SCM、干预、反事实推理）与深度学习模型（如VAE、GNN、ViT、LLM）深度融合。该领域的核心趋势在于，通过发掘数据背后的因果结构来学习不变表征，从而克服机器学习模型在面对分布外数据、伪相关和缺乏可解释性时的固有问题。未来的研究方向将聚焦于更复杂的因果结构学习、反事实样本的高效生成以及因果理论在更多现实任务中的应用。

文献矩阵

下表对核心文献的研究目标、方法、数据集、关键发现及局限性进行了归纳与比较。

文献 研究目标 核心方法 关键发现/贡献 局限性/未来方向
Deng 2025[1] 综述深度学习如何从表征、发现、推断三方面增强因果学习 文献综述 系统回顾了深度学习解决传统因果学习挑战的机制，强调了深度因果学习对推动因果科学的前沿研究和应用至关重要。 作为综述，未提出新的模型或方法，未来工作需在实践中验证和拓展综述中提及的各方向。
Schölkopf 2021[2] 综述因果推断与机器学习的交叉，强调因果表征学习是核心问题 文献综述 提出因果关系可以促进机器学习的迁移与泛化，而机器学习需要从低级数据中发现高级因果变量。 因果表征学习的理论和算法尚不成熟，如何从观测数据中无监督地发现因果变量仍是重大挑战。
Lu 2026[3] 提升脑机接口在分布外场景下的泛化能力 结构因果模型 + 黎曼几何 + 变分信息瓶颈 提出的CD-CMAN模型通过解耦语义和变化因子，学习不变表征，在跨受试者EEG解码中优于基线方法。 方法在特定数据集上验证，其泛化到其他复杂生理信号的能力有待检验。
Liu 2023[4] 解决视觉问答中的跨模态伪相关和事件级推理简化问题 因果干预(前门/后门) + 时空Transformer 提出的CMCIR框架能有效发现视觉-语言因果结构，在事件级视觉问答任务上实现鲁棒的性能。 因果干预操作的设计可能依赖于特定任务，复杂场景下的因果图自动构建仍面临挑战。
Zhou 2025[5] 学习跨域不变表征以用于迁移学习 图自编码器 + 自编码器（GeAE） GeAE能处理非线性因果关系和高维数据，为跨域分类提供有效的特征表示。 模型性能可能依赖于因果结构学习模块的准确性，对未观测混杂因素的鲁棒性未深入探讨。
Hu 2025[6] 构建稳定且可解释的注意力机制 SEAT/SEAT++，约束注意力分布 提出的注意力变体相较于原始注意力，在对抗随机扰动时更稳定，同时保持了可解释性和预测准确性。 定义和实现稳定性的方式可能限制了模型在某些任务上的灵活性。
Zhou 2024[7] 为航空航天产品制造的质量问题提供可信赖的原因分析 结构因果图 + 大语言模型(LLM微调) 提出的CausalKGPT通过注入高质量因果知识，在质量缺陷推理方面比ChatGPT等商业模型更专业可靠。 因果知识图谱的构建和准确性高度依赖于领域专家，模型在完全开放域的问答能力待验证。
Zhang 2026[8] 提升工业过程监控的可靠性和可解释性 空间自注意力 + 图卷积LSTM + 因果图发现 CGSTAE通过学习变量间的不变因果图进行过程监控，在田纳西-伊斯曼过程和空分过程上验证了有效性。 因果图结构学习的三步算法在非常大规模变量下的效率需要进一步优化。
Ma 2025[9] 高效地进行多标签因果特征选择 互信息 + 马尔可夫毯搜索 (MI-MCF) MI-MCF使用互信息替代耗时的条件独立性测试，能自动确定最优特征数量，性能优于对比方法。 方法基于互信息，其对高维离散化或复杂非线性关系数据的有效性有进一步探索空间。
Liang 2025[10] 增强时空预测模型的可解释性与对噪声/不确定性的鲁棒性 动态图扩散模型 + 变分推断 + GCN (DVGNN) DVGNN能动态推断连边概率并重构因果图，在多个数据集上取得优异预测结果，并能反映因果关系和不确定性。 扩散模型的推理过程可能较慢，不适用于极高实时性要求的场景。
Ding 2024[11][12] 提高GCN推荐模型的增量更新效率，同时不牺牲准确性 因果增量图卷积 (IGC) + 碰撞效应蒸馏 (CED) IGC方法能仅使用新数据高效更新模型，并通过CED维持不活跃节点的时效性，实现准确率与速度的兼顾。 CED模块对碰撞变量控制的精确性可能影响不活跃节点表征的估计质量。
Zhu 2024[13] 增强高分辨率遥感图像语义分割在域自适应中的泛化能力 因果特征解耦 + 因果原型对比 + 因果干预 提出的CPCA方法能学习域不变的因果特征，并通过干预生成反事实样本，显著提升了分布外泛化性能。 因果特征与偏置特征的解耦效果直接影响模型性能，该过程可能不稳定或存在信息泄露。
Zhao 2026[14] 在开放世界中学习可迁移的遥感图像不变视觉表征 对象级因果图 + 分区干预(CPI) + 因果表征分离(CRS) PCIR方法通过对象级因果图模拟不同对象受不同环境影响的风格差异，在场景分类和语义分割上提升显著。 模型依赖于目标检测标签来识别对象区域，其性能上限受限于检测器的准确性。
Shi 2024[15] 使遥感图像语义分割更好地利用异构预训练知识 因果公式化 + 逐步干预和重加权 (SIR) SIR方法从因果视角分析预训练偏差，通过干预和重加权减少混杂偏见，提升了多种分割模型的性能。 方法主要面向预训练-微调范式，其能否推广到从头训练或自监督预训练模型有待探索。
Zhang 2025[17] 解决大规模长尾视频动作识别中的伪相关和数据不均衡问题 结构因果模型(SCM) + 因果干预 + 反事实推理(CICR) CICR通过识别并以动量为工具干预混淆因子，结合反事实推理，有效缓解了伪相关和长尾问题，准确率提升显著。 构建的SCM较为通用，对于包含更复杂时空因果结构的视频场景可能不是最优的。
Li 2023[18] 提升Vision Transformer (ViT)的图像识别鲁棒性 结构因果模型 + 因果干预 (Causal-ViT) Causal-ViT通过近似干预消除了全局上下文带来的混杂偏差，能灵活嵌入多种子任务并带来性能提升。 干预是近似的，与do-calculus的精确干预仍有差距，在不同架构上的最优实现方式可能不同。
You 2026[21] 从非结构化数据中学习具有因果效应的表征 三元组网络 + 图注意力 + 结构因果模型 (CRLTGA) CRLTGA能聚合上下文因果信息并利用三元组损失优化表征空间，干预实验和下游任务证明了其有效性。 模型复杂度较高，涉及VAE、GAN和GAT的联合训练，训练稳定性和调参难度是需要关注的问题。
You 2026[22] 在VAE中实现考虑因果效应传递的解耦表征学习 非线性SCM + 判别器层次特征损失 + 图注意力网络(DRLCET) DRLCET能更新因果结构并通过GAT传输因果效应，生成更真实的干预样本，优于当前最优方法。 因果结构的可辨识性在理论上有一定保障，但实践中对复杂真实数据的适用程度仍需检验。
Arjovsky 2020[40] 学习跨不同训练分布的不变相关性，以实现分布外泛化 不变风险最小化 (IRM) IRM从理论上将不变性与数据背后的因果结构联系起来，并在实验中被证明能引导模型找到更优的分布外解。 原始IRM在复杂高维数据和有限训练环境数时表现可能不稳定，寻找最优的不变表征是困难的优化问题。
Ahuja 2022[41] 探索干预数据在因果表征学习中的力量与保证 干预数据几何分析 证明了在完美干预下可辨识潜在因果因素，在不完美干预下可实现块仿射辨识，揭示了干预数据的独特价值。 该方法要求可获得不同干预下的数据，这在许多实际应用（如纯观测性研究）中是难以满足的强假设。
Yang 2021[42] 学习含有因果结构的可解耦表征 神经结构因果模型 + VAE (CausalVAE) CausalVAE能发现变量间的因果有向无环图，学习的表征语义可解释，并能生成反事实数据。 因果层的可辨识性依赖于线性SCM假设和一些技术条件，对非线性可辨识性的推广是重要的研究方向。
Common Features 绝大多数研究旨在提升模型的泛化性、鲁棒性或可解释性。核心方法是将因果推断的原则（如不变性、干预）与深度学习模型（如VAE、GNN、Transformer）相结合。研究重心从关联驱动的预测转向因果驱动的理解与决策。
Distinctive Features 具体任务多样（CV、NLP、推荐、工业）。因果的作用方式不同：有的用于表征学习（学习不变因果特征），有的用于结构发现（构建因果图），有的用于去偏/推理（干预、反事实）。数据假设不同：部分假设仅有观测数据，部分依赖干预数据。模型基础结构各异：基于VAE、GNN、Transformer、Mamba等不同架构。
